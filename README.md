1. Instalar no terminal "npm install"
2. Acessar a pasta deck em um cmd separado, depois dá um "node server.js"
3. Abrir o front em outro terminal, usando "npm run dev"


Proposito: O sistema tem proposito de permitir consulta NCM dos produtos nacionais brasileiros, através da API Siscomex. A busca pode ser feita utilizando o nome do tipo do produto, ou diretamente pelo codigo do tipo do mesmo. É permitido tambem, favoritar.
